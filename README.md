 # 🐍 Retro Snake Game

A modern Snake Game built using Kotlin and Jetpack Compose with Google AI Studio.

## Features
- Classic Snake gameplay
- Score tracking
- Modern UI
- Android support

## How to Run
1. Download the APK from the Releases section.
2. Install it on your Android device.
3. Enjoy the game!
